# Standards Directory

هذا المجلد يحتوي على المعايير الإلزامية للمنصة.

## Standards

### UX Helper Kit Standard

**File**: `UX_HELPER_KIT_STANDARD.md`

**Status**: ✅ MANDATORY

**Scope**: جميع الواجهات (Apps, Dashboards, Web)

**Description**: معيار إلزامي لاستخدام مكونات المساعدة السياقية الموحدة.

**Quick Reference**: `shared/ui/CHECKLIST.md`

## Enforcement

جميع المعايير في هذا المجلد **إلزامية**. أي PR لا يلتزم بالمعايير سيتم رفضه حتى يتم التصحيح.

## Adding New Standards

عند إضافة معيار جديد:

1. أنشئ ملف `.md` في هذا المجلد
2. أضف مرجع في هذا README
3. أضف المعيار إلى `.github/Cursor/rules/` إذا لزم
4. أضف CI/CD gates إذا لزم
